 >천안 독립기념관에서 개최되는 '2023 천안 K-컬처 박람회' <br/>

## 기술 스택

**클라이언트:** HTML(with CDN), ionIcons, TailwindCSS, Flowbite, DaisyUI, Devices.css

**AR구현:** MindAR.js, Aframe.js, Three.js, Three-ar.js, Model-Viewer

**서버 빌드:** Vercel, CloudFlare

<br/>


## 특징

- 얼굴 추적 AR
